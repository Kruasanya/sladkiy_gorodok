# Применение методов машинного обучения и финансового анализа в условиях реального бизнеса

## Контекст
Есть предприятие которые производит пищевую продукцию: печенье, сладости, выпечку. Продает его в крупные сети дискаунтеров, магазинов низких цен в Москве и Московской области, Тульской области - Светорфор, Маяк. А также на торговые рынки Москвы. У предприятия два места производства, которые связаны между собой и дополняют друг друга.

## Проблема
Бизнес работает, функционирует, платит зарплаты, кредиты, долги, но при этом не приносит свободного дохода для вывода.

## Цель
Провести финансовый и операционный аудит, понять:
1. Как повысить операционную прибыль и увеличить доход для вывода средств из бизнеса
2. Определить слабые места бизнеса, понять какие места можно оптимизировать
3. Составить план и четкие шаги к действиям по:
    - увеличению прибыли
    - автоматизации процессов
