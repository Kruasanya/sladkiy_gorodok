# Данные

Эта папка предназначена для всех работ с данными проекта: здесь лежат сырые выгрузки, пайплайны обработки и готовые аналитические таблицы.

## Где что хранится

| Папка | Что хранится |
|---|---|
| `Продажи по контрагентам` | Сырые Excel-отчеты по продажам. |
| `Продажи по контрагентам (по оплате)` | Сырые Excel-отчеты по оплатам. |
| `Выписки и данные` | Сырые банковские выписки и связанные исходные файлы. |
| `Пайплайны обработки данных` | Python-скрипты, которые превращают сырые файлы в аналитические таблицы. |
| `Итоговые таблицы` | Готовые таблицы в форматах Parquet, CSV и XLSX. |

## Как работать

1. Сырые файлы кладутся в соответствующую исходную папку.
2. Нужный скрипт запускается из `Пайплайны обработки данных`.
3. Результат появляется в `Итоговые таблицы`.

Основной формат для анализа через pandas - `.parquet`.
